Dart는 플랫폼에 상관없이 빠른 앱을 개발하기 위한 클라이언트에 최적화된 언어입니다.
UI를
위한 언어
유저 인터페이스 작성에 전문화된 프로그래밍 언어로 개발해보세요.
생산성 높은
개발
반복적인 변경이 있을 때, 실행 중인 앱에서 즉시 결과를 볼 수 있는 hot reload를 사용할 수 있습니다.
모든 플랫폼에서
빠릅니다
모바일, 데스크탑, 백엔드를 위한 ARM & x64 기계어 (machine code)로 컴파일됩니다. 또는 web에서 사용 가능한 JavaScript로 컴파일도 가능합니다.
UI를 위한 언어
  • 격리 기반 동시성과 쌍을 이루고 이벤트 기반 코드가 포함된 유저 인터페이스를 작성할 수 있는 성숙하고 완전한 async-await를 살펴보세요.
  • 컬렉션을 확장하기 위한 spread operator 나 각 플랫폼 별로 UI를 커스터마이징하기 위한 collection if 같이, 유저 인터페이스를 만드는 데 필요한 기능들을 포함한 프로그래밍 언어입니다.
  • 친숙한 문법으로 배우기 쉬운 프로그래밍 언어입니다.
생산성 높은 개발
  • 소스 코드를 반복적으로 수정해야할 때 실행 중인 앱에서 바로 변화를 확인할 수 있는 hot reload를 사용해보세요.
  • 풍부한 정적 검사 도구를 사용해 유연한 타입 시스템을 사용해 코드를 작성할 수 있습니다. 도구 설정하기를 살펴보세요.
  • 여러분이 선택한 코드 에디터에서 프로파일링, 로깅, 디버깅 을 시작하세요!
모든 플랫폼에서 빠릅니다.
  • 즉시 앱을 시작하기 위해 네이티브 머신 코드로 이루어진 AOT 컴파일 앱을 만들 수 있습니다.
  • 완전하고, 성숙하고, 빠른 JavaScript 컴파일러를 사용해서 Web을 타겟으로 한 앱을 만들 수 있습니다.
  • 앱을 지원하기 위한 백엔드 코드를 하나의 프로그래밍 언어를 사용해서 작성할 수 있습니다.
